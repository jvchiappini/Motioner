### Playback
- `Space` — Play/Pause
- `Home` — Jump to start
- `End` — Jump to end
- `Left Arrow` — Previous frame
- `Right Arrow` — Next frame

### Timeline
- `+` / `Scroll Up` — Zoom in
- `-` / `Scroll Down` — Zoom out

### General
- `Ctrl+S` — Save project
- `Ctrl+O` — Open project
- `Ctrl+E` — Export
