## Key Features

### Fast & Responsive UI
Built with [egui](https://github.com/emilk/egui), Motioner provides an immediate-mode GUI that's both intuitive and performant.

### Timeline-Based Editing
Work with a professional timeline editor that gives you precise control over your animations.

### Frame-by-Frame Export
Export your animations as PNG sequences or directly encode to MP4 using FFmpeg.

### GPU-Ready Architecture
Modular design prepared for GPU-accelerated rendering with [wgpu](https://wgpu.rs/).
